Moneta MCP Server

Connect your AI assistant to your personal finance data.

What is Moneta?

Moneta is a personal finance tracking app that lets you manage accounts, record transactions, set budgets, and analyze spending. The Moneta MCP server allows AI assistants like Claude, ChatGPT, and Gemini to interact with your financial data through natural language.

Moneta is a budgeting and record-keeping tool. It records entries in your personal ledger. It does not connect to banks, transfer money, or execute financial transactions of any kind.

Getting Started

Claude (Web & Desktop)

  1. Go to Settings → Connectors → Add custom connector
  2. Paste the server URL: https://mcp.coinstack.cloud/mcp
  3. Complete the OAuth login with your Moneta account
  4. Start asking questions about your finances

Claude Code

Add to your MCP settings:

{
  "mcpServers": {
    "moneta": {
      "type": "streamable-http",
      "url": "https://mcp.coinstack.cloud/mcp"
    }
  }
}

Authentication

Moneta uses OAuth 2.1 with PKCE and dynamic client registration. When you connect via an AI assistant, the OAuth flow is handled automatically:

  1. The AI client registers itself via /oauth/register
  2. You are redirected to the Moneta login page
  3. After login, you authorize the connector
  4. The AI assistant receives a JWT access token for subsequent requests

Discovery endpoints:

  • /.well-known/oauth-authorization-server — OAuth metadata
  • /.well-known/oauth-protected-resource — resource metadata
  • /.well-known/mcp.json — tool manifest

Usage Examples

Check your balances

"What are my account balances?"
→ calls list_accounts
→ returns all accounts with names, types, currencies, and balances

Track spending

"How much did I spend on food this month?"
→ calls get_spending_summary
→ returns total expenses with category breakdown

Log an expense

"I spent $45 at Chipotle today from my checking account"
→ calls list_categories, then add_transaction
→ records $45 expense in Food & Dining from Checking

Check budgets

"Am I over budget on anything?"
→ calls check_budget_alerts
→ lists categories exceeding or approaching their monthly limit

Set a budget

"Set my entertainment budget to $250 for May"
→ calls list_categories, then set_budget
→ confirms Entertainment budget updated to $250 for May 2026

Multi-Currency Support

Moneta supports 10 currencies: USD, EUR, GBP, UAH, PLN, CZK, CHF, CAD, AUD, JPY. Exchange rates are fetched from the National Bank of Ukraine and cached for 1 hour. Balances can be viewed in any supported currency.

Troubleshooting

  • OAuth login fails: Make sure you have an active Moneta account at app.coinstack.cloud.
  • Tool returns an error: Check that your access token hasn't expired. Reconnect the connector to refresh it.
  • Missing data: Moneta only shows data you've entered. It does not sync with banks or import transactions automatically.
